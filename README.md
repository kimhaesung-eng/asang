## GitHub Pages 배포

이 리포지토리의 `asang-main` 폴더를 GitHub Pages로 배포하도록 자동화 워크플로를 추가했습니다.

- 코드를 `main` 브랜치에 푸시하면 자동으로 배포됩니다.
- 배포 후에는 아래 URL로 접속할 수 있습니다:
  - `https://kimhaesung-eng.github.io/asang/`

> GitHub Pages가 활성화되면, `asang-main` 폴더 내부의 `index.html`이 루트 페이지로 사용됩니다.

아직 Pages가 활성화되지 않았다면, GitHub 리포지토리의 Settings > Pages에서 기본 브랜치와 경로를 확인해 주세요.
